MCMS
=============

MCMC Simulations of Protein Concentration and Modification Occupancy Ratios from MS Experiments

Installation
-------

```R
install.packages("devtools")
devtools::install_github("xrobin/MCMS")
```

Getting started
-------

```R
library(MCMS)
vignette("introduction", "MCMS")
```
